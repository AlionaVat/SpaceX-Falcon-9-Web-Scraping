# SpaceX Falcon 9 Web Scraping

Web scraping project that extracts historical Falcon 9 launch records from Wikipedia and prepares a structured dataset for further analysis.

This notebook is part of the **IBM Data Science Professional Certificate – SpaceX Falcon 9 Capstone Project**.

---

## Overview

The project demonstrates how publicly available launch information can be collected directly from Wikipedia using Python web scraping techniques.

The notebook downloads the launch history page, parses HTML tables with **BeautifulSoup**, extracts mission details, performs basic data cleaning, and converts the collected information into a structured dataset for the subsequent stages of the data science workflow.

---

## Project Objectives

- Retrieve Falcon 9 launch history from Wikipedia
- Parse HTML content using BeautifulSoup
- Extract launch records into a structured format
- Clean and organize the collected data
- Prepare the dataset for data wrangling and exploratory analysis
